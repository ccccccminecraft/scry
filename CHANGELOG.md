# Changelog

## v0.1.3 (2026-03-22)

### 変更

- MTGA カードデータベース同期をファイルアップロード方式に変更
  （Electronメインプロセスからバックエンドに直接 HTTP POST することで Windows パス解決の問題を回避）
- `build.py` を廃止し、手動ビルド手順に戻す

## v0.1.1 (2026-03-22)

### バグ修正

- MTGAカードデータベースの同期が本番環境（exe）で失敗する問題を修正

## v0.1.0 (2026-03-22)

初回リリース。

### 対応フォーマット

- **MTGO**（Magic: The Gathering Online）: `.dat` ログファイルのインポート
- **MTGA**（Magic: The Gathering Arena）: Surveil 連携による対戦データのインポート

### 機能

#### インポート
- MTGO `.dat` ファイルの単体インポート・フォルダ一括インポート
- MTGA ログの自動インポート（Surveil 常駐ツールとの連携）
- バックグラウンド定期スキャンによる自動インポート
- インポート履歴の管理

#### 対戦履歴
- 対戦一覧の閲覧（フォーマット・デッキ・対戦相手・日付で絞り込み）
- ゲーム詳細・アクションログの閲覧
- 対戦履歴へのデッキバージョン紐づけ

#### 統計ダッシュボード
- 勝率・先手後手・マリガン率の集計
- デッキ別勝率グラフ（最低マッチ数フィルター付き）
- 対戦相手カード統計
- フォーマット・プレイヤー・デッキ・期間での絞り込み

#### デッキ管理
- デッキ登録・バージョン管理
- `.dek` ファイルインポート
- デッキ代表カードの設定（サムネイル表示）
- 対戦履歴への一括適用

#### AI 分析
- Claude API を使ったデッキ分析チャット
- 対戦履歴・デッキリストを含むデータエクスポート

#### 設定・管理
- MTGO / MTGA ログフォルダの設定
- MTGA カードデータベース同期（`Raw_CardDatabase_*.mtga` から取得）
- SQLite データベースのバックアップ・リストア
- データ削除（全削除・期間削除・完全リセット）
- デフォルトプレイヤー設定
- 初回セットアップウィザード
