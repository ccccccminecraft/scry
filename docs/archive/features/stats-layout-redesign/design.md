## 対戦相手デッキ別勝率 仕様

### 表示内容
- 現在のフィルター条件に合致する試合を対象に、対戦相手のアーキタイプ別の集計を行う
- 試合数が多い順に上位 10 件を表示
- データなし時: 「データなし」テキスト表示

### 形式
カード統計テーブルと同じスタイルの**テーブル**として表示する。

| 列 | 内容 |
|----|------|
| アーキタイプ名 | 対戦相手のアーキタイプ（`deck_name`） |
| 対戦数 | その相手との試合数 |
| 勝率 | 選択プレイヤーの勝率（例: `60.0%`） |

- 列ヘッダークリックでソート可能（対戦数 / 勝率）
- デフォルトソート: 対戦数 降順

## 削除対象

| 対象 | 理由 |
|------|------|
| `FirstSecondChart.vue` | 使用箇所がなくなるため削除 |
| `DeckStatsChart.vue` | 使用箇所がなくなるため削除 |
| `OpponentDeckStatsChart.vue` | チャートではなくテーブルで実装するため不要 |
| `StatsResponse.first_play_win_rate` | フロントエンド型から削除（バックエンドは互換性のため残してもよい） |
| `StatsResponse.second_play_win_rate` | 同上 |
| `StatsResponse.deck_stats` | フロントエンド型から削除（バックエンドは互換性のため残してもよい） |

> バックエンドの `deck_stats` / `first_play_win_rate` / `second_play_win_rate` フィールドはレスポンスに残したままでよい（削除しても問題ないが、今回は変更対象外とする）。

## 変更しない仕様

- サマリーカード（総試合数 / 勝率 / 平均ターン数 / マリガン率）はそのまま
- 勝率推移チャートはそのまま（直近 N 件 / 全試合 の切り替えも維持）
- カード統計（折りたたみ）はそのまま
