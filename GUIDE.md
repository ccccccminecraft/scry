## 1. インストール

### 動作環境

- Windows 10 / 11（64bit）

### インストール手順

1. [Releases](../../releases) ページから最新の `ScrySetup.exe` をダウンロード
2. `ScrySetup.exe` を実行
3. インストール先を確認して「インストール」をクリック
4. 完了後、スタートメニューまたはデスクトップのショートカットからScryを起動

### SmartScreen 警告について

初回実行時に「WindowsによってPCが保護されました」という警告が表示される場合があります。

1. 「詳細情報」をクリック
2. 「実行」をクリック

### データの保存場所

対戦データ・設定はすべて以下のフォルダに保存されます。

```
C:\Users\<ユーザー名>\AppData\Roaming\Scry\mtgo.db
```

アンインストールしてもこのフォルダは残るため、再インストール後もデータは引き継がれます。

### アップデート

新しいバージョンの `ScrySetup.exe` を実行するだけでアップデートできます。データはそのまま保持されます。

### アンインストール

Windowsの「設定」→「アプリ」→「Scry」から削除できます。データを完全に削除する場合は、上記の保存フォルダも手動で削除してください。

## 2. 初回セットアップウィザード

アプリを初めて起動すると、セットアップウィザードが自動的に表示されます。約2分で基本設定を完了できます。

```
[Welcome] → [ゲーム選択] → [インポート設定] → [AI分析設定] → [完了]
```

### Welcome

アプリの概要が表示されます。「セットアップを開始する」をクリックして進んでください。「スキップ」をクリックすると設定を省略して後から行うこともできます。

### Step 1: ゲーム選択

使用するゲームを選択します（複数選択可）。

- **MTGO** — ログファイル（`.dat`）からインポート
- **MTGA（Surveil 経由）** — Surveilが出力するJSONファイルからインポート

### Step 2: インポート設定

Step 1の選択に応じた設定項目が表示されます。

| ゲーム | 設定項目 |
|--------|---------|
| MTGO | 登録フォルダ（ログが保存されるフォルダ）・自動インポートON/OFF |
| MTGA | Surveilフォルダ（matches/フォルダ）・MTGAインストールフォルダ・自動インポートON/OFF |

すべての項目は省略可能です。後から **設定** 画面や **インポート** 画面で変更できます。

### Step 3: AI分析設定（任意）

Anthropic の API キーを設定します。AI分析機能が不要な場合はスキップできます。

### 完了

設定内容のサマリーが表示されます。「ログをインポートする」または「設定を確認する」をクリックして次のステップへ進んでください。

### ウィザードの再実行

**設定** 画面の「セットアップウィザードを再実行」ボタンからいつでも再実行できます。

## 3. 対戦ログのインポート

**インポート** 画面では MTGO と MTGA（Surveil）のログを取り込みます。画面上部のタブで切り替えてください。

---

### MTGO

#### ログファイルの場所

MTGOのログは通常、以下のフォルダに保存されています。

```
C:\Users\<ユーザー名>\AppData\LocalLow\Wizards of the Coast\Magic Online\Logs
```

また、MTGOインストーラーによっては以下の場所に保存されることがあります。

```
C:\Users\<ユーザー名>\AppData\Local\Apps\2.0\（サブフォルダ内）
```

#### インポート方法

**① 登録フォルダから取り込む（おすすめ）**

ログフォルダを一度登録しておくと、**⚡ 新着を取り込む** ボタン1つで前回インポート以降の新しいファイルだけを自動取り込みできます。

初回登録手順：
1. インポート画面（MTGOタブ）の「フォルダを登録する」をクリック
2. MTGOのログフォルダを選択
3. 以降は ⚡ ボタンをクリックするだけで新規ログが取り込まれます

**② ドラッグ＆ドロップ**

`.dat` ファイルをインポート画面にドラッグして放すと即座に取り込みが始まります。

**③ フォルダから一括で取り込む**

「📁 フォルダから一括で取り込む」ボタンからフォルダを指定すると、フォルダ内の `.dat` ファイルを一覧表示します。チェックを入れたファイルだけを取り込めます。

---

### MTGA（Surveil 経由）

MTGAのログは [Surveil](https://github.com/nbusseneau/surveil)（外部ツール）を通じてJSONファイルとして出力されます。事前にSurveilをインストールして対戦ログを収集しておく必要があります。

#### インポート方法

**① 登録フォルダから取り込む（おすすめ）**

SurveilのJSONが保存されるフォルダ（matches/フォルダ）を登録すると、**⚡ 新着を取り込む** ボタンで未取り込みのファイルのみを取り込めます。

**② 単体ファイルを取り込む**

「ファイルを選択」でSurveilのJSONファイルを個別に取り込みます。

**③ フォルダから一括で取り込む**

「📁 フォルダから一括で取り込む」ボタンからSurveilのフォルダを指定すると、フォルダ内のJSONファイルをまとめて取り込めます。

---

### 自動インポート

登録フォルダを設定し、**設定** 画面で自動インポートを有効にすると、バックグラウンドで定期的に新着ログを自動取り込みします。スキャン間隔はデフォルト30秒で変更可能です。

## 4. 対戦履歴の閲覧

### 対戦一覧

**対戦履歴** 画面では取り込んだ全対戦が一覧表示されます。

- 対戦日時・使用デッキ・相手デッキ・勝敗・フォーマットを確認できます
- 列ヘッダーをクリックして並び替え可能です

### 対戦詳細

一覧の行をクリックすると詳細画面に移動し、ゲームごとのアクションログを確認できます。

---

## 5. 統計ダッシュボード

**統計** 画面では対戦データをグラフで確認できます。

### フィルター

画面上部のフィルターで表示するデータを絞り込めます。

| フィルター | 内容 |
|-----------|------|
| プレイヤー | 分析対象のプレイヤー名 |
| 使用デッキ | 自分が使用したデッキ |
| 相手デッキ | 対戦相手のデッキ |
| フォーマット | モダン / パイオニア等 |
| 対戦日時 | 期間を指定して絞り込み |

### 表示内容

- **勝率サマリー** — 総試合数・勝率・平均ターン数・マリガン率
- **勝率推移グラフ** — 直近試合の勝率推移
- **先手 / 後手勝率** — 先手・後手それぞれの勝率
- **デッキ別勝率** — 使用デッキごとの勝率比較
- **カード統計（選択プレイヤー）** — 自分が使用したカードの使用回数・登場ゲーム数・勝率
- **カード統計（対戦相手）** — 相手が使用したカードの使用回数・登場ゲーム数・自分の勝率

カード統計セクションはデフォルトで折りたたまれています。タイトル行をクリックすると展開できます。

## 6. AI分析

**AI分析** 画面では、取り込んだ対戦データをもとにAIとチャット形式で分析できます。

### 基本的な使い方

1. **プレイヤー** を選択
2. 必要に応じてフィルター（対戦相手・デッキ・期間等）を設定
3. 「会話を開始する」をクリック
4. AIが対戦データの概要を提示するので、質問を入力して分析を深めます

### 質問ボタン

画面下部に表示される質問ボタンをクリックすると、定型の質問をワンクリックで送信できます。質問の内容は [プリセット管理](#8-プリセット管理) でカスタマイズできます。

### フィルターの活用

特定の対面やデッキに絞って分析したい場合は、フィルターを設定してから会話を開始してください。AIはフィルター後のデータをもとに回答します。

### 会話の管理

- 過去の会話は左のサイドバーから選択して参照できます
- 「会話をリセット」で新しい分析セッションを開始できます
- 「続きから会話する」で過去の会話に続きを追加できます

## 7. エクスポート

**AI用エクスポート** 画面では、対戦データをMarkdownファイルとして書き出せます。外部のAIサービス（ChatGPT等）に貼り付けて分析する用途に使えます。

### フィルター

プレイヤー・対戦相手・デッキ・フォーマット・対戦日で出力対象を絞り込めます。フィルターを変更すると対象件数がリアルタイムで表示されます。

### 出力内容

| 詳細レベル | 内容 | 推奨用途 |
|-----------|------|---------|
| サマリーのみ | 統計サマリー・デッキ別勝率・カード統計 | 全体傾向の把握 |
| マッチ一覧あり | ＋各マッチの基本情報・ゲーム結果 | デッキ・マッチアップ分析 |
| アクション詳細あり | ＋各ゲームのターン別アクション | プレイング改善 |

### 件数上限

「制限なし（全件）」にチェックを入れると全件を出力します。チェックなしの場合は直近N件に絞り込めます。アクション詳細を含む場合はファイルサイズが大きくなるため、件数を絞ることをおすすめします。

## 8. プリセット管理

**プリセット管理** 画面では、AI分析で使うテンプレートと質問セットを管理します。

### プロンプトテンプレート

AIへの指示文（システムプロンプト）を定義します。分析の方針や回答スタイルをカスタマイズできます。

- `{player_name}` — プレイヤー名に置換されます
- `{stats_text}` — 対戦統計データに置換されます

### 質問セット

AI分析画面に表示するクイック質問ボタンの内容を定義します。よく使う質問を登録しておくと便利です。質問は上下ボタンで並び替えられます。

### 既定の設定

テンプレート・質問セットともに「既定にする」を設定すると、AI分析画面を開いたときに自動で選択されます。

---

## 9. デッキ管理

**デッキ管理** 画面では、使用デッキの構成を登録・管理します。

### デッキの登録

「＋ デッキを追加」からデッキ名を入力して新規デッキを作成できます。MTGAからインポートした対戦はデッキが自動的に登録されます。

### デッキバージョン

1つのデッキに複数のバージョン（調整前・調整後など）を持てます。各バージョンにカードリストを登録できます。

カードリストの登録方法：
- テキスト入力（カード名を1行ずつ）
- `.dek` ファイルのインポート（MTGOのデッキファイル）

### 代表カード

デッキのサムネイルとして代表カードを設定できます。統計画面のデッキサマリーに表示されます。

### 対戦履歴との紐づけ

**対戦履歴** の詳細画面から、その対戦で使用したデッキバージョンを手動で紐づけられます。

---

## 10. デッキ定義

**デッキ定義** 画面では、デッキ名の表記を統一するルールを管理します。

インポートしたログに含まれるデッキ名が複数の表記で登録されている場合（例：「4cRhinos」と「4C Rhinos」）、ここで統一名を設定することで統計やAI分析の精度が向上します。

## 11. 設定

**設定** 画面では以下を管理します。

### アプリケーション

| 操作 | 内容 |
|------|------|
| 再読み込み | 表示がおかしい場合や設定を反映させたい場合 |
| セットアップウィザードを再実行 | 初回セットアップウィザードをもう一度表示 |

### デフォルトプレイヤー

統計・AI分析・エクスポート画面を開いたときに自動で選択されるプレイヤー名を設定します。

### 絞り込みの最低試合数

プルダウンの候補に表示するプレイヤーおよびデッキの最低試合数を設定します。0で全件表示します。

### 自動インポート

登録フォルダを定期スキャンして新しいログを自動取り込みします。

| 項目 | 内容 |
|------|------|
| 有効/無効 | 自動インポートのON/OFF |
| スキャン間隔 | スキャンの間隔（秒）。デフォルト30秒 |

### カード名データベース（MTGA）

MTGAのカード名解決に使うデータベースを同期します。MTGAのアップデート後に同期することで新カードを正しく認識できます。アプリ起動時にファイルの更新を検知した場合は自動で同期します。

| 操作 | 内容 |
|------|------|
| 参照... | MTGAのインストールフォルダを選択（例: `C:\Program Files\Wizards of the Coast\MTGA`） |
| MTGAデータを同期 | 選択フォルダ内の `Raw_CardDatabase_*.mtga` を読み込んでカード名キャッシュを更新 |

### Anthropic API キー

AI分析機能で使用するAPIキーを登録します。APIキーは [Anthropic Console](https://console.anthropic.com/) で取得できます。ローカルのデータベースに保存されます。

### データベース

| 操作 | 内容 |
|------|------|
| バックアップをダウンロード | データベースファイルをダウンロード |
| リストア | バックアップファイルからデータを復元 |

### データ削除

| 操作 | 内容 |
|------|------|
| 全試合データを削除 | 設定・デッキ定義を残してすべての試合を削除 |
| 期間指定で削除 | 指定した期間の試合のみ削除 |
| 完全リセット | すべてのデータ（試合・設定・デッキ定義）を完全削除 |

## トラブルシューティング

### ログが取り込めない

- `.dat` ファイルが正しいMTGOのログファイルか確認してください
- すでに取り込み済みのファイルは重複インポートされません

### 登録フォルダから新しいファイルが検出されない

- インポート画面で正しいフォルダが登録されているか確認してください
- 登録フォルダを変更した場合は、再度フォルダを登録してください

### MTGAのカード名が正しく表示されない

- **設定** 画面の「カード名データベース」でMTGAインストールフォルダを設定し、「MTGAデータを同期」を実行してください
- MTGAのアップデート後は再同期が必要です（アプリ起動時に自動検知します）

### AI分析が動作しない

- **設定** 画面でAPIキーが登録されているか確認してください
- APIキーの残高・利用制限を [Anthropic Console](https://console.anthropic.com/) で確認してください

### バックエンドに接続できない（ホーム画面でエラー表示）

- アプリを再起動してください
- アプリを完全に終了してから再度起動してください
